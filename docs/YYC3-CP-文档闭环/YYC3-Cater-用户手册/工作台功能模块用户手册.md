## 📖 模块简介

### 欢迎使用YYC³工作台

YYC³工作台是餐饮行业智能化平台的管理核心，为您提供一站式的数据监控和操作入口。通过工作台，您可以：

- 📊 **实时监控**：查看今日订单、销售额、客户数等关键数据
- ⚡ **快速操作**：通过快捷入口快速访问常用功能
- 📝 **任务管理**：管理待办事项，确保重要任务不遗漏
- 🔔 **即时通知**：接收系统通知，及时了解重要信息

### 适用角色

- **店长**：查看整体运营数据，管理门店日常事务
- **收银员**：快速创建订单，处理支付事务
- **厨师长**：查看订单状态，管理厨房工作
- **客服人员**：处理客户咨询，管理客户信息

## 🚀 快速开始

### 登录系统

1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击"登录"按钮
4. 进入工作台首页

### 首次使用

首次登录后，系统会引导您完成以下设置：

1. **个人信息设置**：完善您的个人信息
2. **通知设置**：配置通知偏好
3. **快捷操作设置**：添加常用功能到快捷操作栏
4. **待办事项设置**：设置待办事项提醒

### 界面导航

工作台主要分为以下几个区域：

- **顶部导航栏**：包含系统Logo、用户信息、全局操作
- **左侧菜单栏**：包含所有功能模块的导航入口
- **主内容区**：显示当前页面的主要内容
- **右侧面板**：显示快捷操作、待办事项、通知中心

## 📊 实时数据看板

### 核心指标

工作台顶部显示四个核心指标卡片：

| 指标 | 说明 | 更新频率 |
|------|------|----------|
| 总收入 | 今日营业总收入 | 实时 |
| 订单数 | 今日订单总数 | 实时 |
| 平均客单价 | 平均每单金额 | 每小时 |
| 客户数量 | 今日客户总数 | 每小时 |

### 查看指标详情

1. 点击指标卡片，查看详细数据
2. 在弹出的对话框中，选择时间范围
3. 查看历史趋势和对比分析
4. 导出数据报表

### 营收趋势图

营收趋势图展示不同时间维度的营收变化：

- **日视图**：查看每日营收变化
- **周视图**：查看每周营收变化
- **月视图**：查看每月营收变化

**操作步骤**：

1. 在图表右上角选择时间维度（日/周/月）
2. 悬停在图表上查看具体数据
3. 点击图例隐藏/显示数据系列
4. 点击"下载"按钮导出图表

### 订单状态分布

订单状态分布图展示当前订单的状态：

- **待处理**：新订单，等待处理
- **制作中**：订单正在制作
- **已完成**：订单已完成
- **已取消**：订单已取消

**操作步骤**：

1. 切换"实时/历史"开关查看不同数据
2. 点击图例查看特定状态的订单
3. 悬停在饼图上查看详细数据

### 菜品销售排行

菜品销售排行展示热销菜品：

- **今日排行**：今日销售最多的菜品
- **本周排行**：本周销售最多的菜品
- **本月排行**：本月销售最多的菜品

**操作步骤**：

1. 在下拉菜单中选择时间范围
2. 查看菜品名称、销售数量、销售金额
3. 点击菜品名称查看详细信息

### 客流量分析

客流量分析展示不同时间段的客流量：

- **时段分布**：每小时客流量
- **高峰时段**：客流量最大的时间段
- **低谷时段**：客流量最小的时间段

**操作步骤**：

1. 查看柱状图了解客流量分布
2. 悬停在柱子上查看具体数据
3. 根据客流量调整人员安排

### 实时运营监控

实时运营监控展示当前系统状态：

- **实时订单**：最新订单列表
- **厨房状态**：厨房工作状态
- **桌台状态**：桌台使用情况

**操作步骤**：

1. 查看实时订单列表
2. 点击"查看详情"查看订单详细信息
3. 根据订单状态进行相应操作

## ⚡ 快捷操作

### 什么是快捷操作

快捷操作是常用功能的快速入口，帮助您快速访问常用功能，提高工作效率。

### 默认快捷操作

系统默认提供以下快捷操作：

| 快捷操作 | 说明 | 跳转路径 |
|----------|------|----------|
| 新建订单 | 创建新订单 | /orders/new |
| 菜品管理 | 管理菜品信息 | /menu/items |
| 客户查询 | 查询客户信息 | /customers/list |
| 查看报表 | 查看数据报表 | /reports/sales |
| 支付管理 | 管理支付交易 | /payment/transactions |
| 系统监控 | 监控系统状态 | /system/monitoring |
| 预约管理 | 管理客户预约 | /reservations |
| 消息中心 | 查看系统消息 | /messages |

### 使用快捷操作

**操作步骤**：

1. 在快捷操作区域找到需要的操作
2. 点击操作卡片
3. 系统自动跳转到对应页面

### 自定义快捷操作

**添加快捷操作**：

1. 点击"添加快捷操作"按钮
2. 填写操作信息：
   - 操作名称：输入快捷操作的名称
   - 操作描述：输入快捷操作的描述
   - 选择图标：从图标列表中选择合适的图标
   - 选择颜色：选择快捷操作卡片的颜色
   - 跳转路径：输入点击后跳转的页面路径
3. 点击"确定"保存

**编辑快捷操作**：

1. 右键点击要编辑的快捷操作
2. 在弹出的菜单中选择"编辑"
3. 修改快捷操作信息
4. 点击"确定"保存

**删除快捷操作**：

1. 右键点击要删除的快捷操作
2. 在弹出的菜单中选择"删除"
3. 确认删除操作

### 快捷操作设置

**打开设置**：

1. 点击快捷操作区域右上角的设置图标
2. 在弹出的对话框中进行设置

**显示设置**：

- 显示徽章：在快捷操作卡片上显示徽章（待处理数量）
- 显示描述：在快捷操作卡片上显示描述文字

**布局设置**：

- 网格布局：以网格形式显示快捷操作
- 列表布局：以列表形式显示快捷操作

**自定义操作管理**：

- 查看所有自定义快捷操作
- 删除不需要的自定义快捷操作

### 快捷操作排序

**拖拽排序**：

1. 按住快捷操作卡片
2. 拖动到目标位置
3. 松开鼠标完成排序

**固定/取消固定**：

1. 右键点击快捷操作
2. 选择"固定"或"取消固定"
3. 固定的快捷操作会始终显示在前面

## 📝 待办事项

### 什么是待办事项

待办事项帮助您管理日常工作任务，确保重要任务不遗漏。

### 查看待办事项

待办事项列表显示所有待办事项，包括：

- 待办标题
- 待办描述
- 优先级（高/中/低）
- 分类
- 截止日期
- 状态（待完成/进行中/已完成）

### 添加待办事项

**操作步骤**：

1. 点击待办事项区域右上角的"+"按钮
2. 填写待办信息：
   - 标题：输入待办事项的标题（必填）
   - 描述：输入待办事项的详细描述
   - 优先级：选择优先级（高/中/低）
   - 分类：选择待办分类（工作/订单/客户/财务/系统/其他）
   - 截止日期：选择待办事项的截止日期
   - 提醒：是否需要提醒
3. 点击"确定"保存

### 编辑待办事项

**操作步骤**：

1. 点击待办事项的编辑按钮
2. 修改待办信息
3. 点击"确定"保存

### 完成待办事项

**操作步骤**：

1. 点击待办事项左侧的复选框
2. 待办事项标记为已完成
3. 已完成的待办事项会显示删除线

### 删除待办事项

**操作步骤**：

1. 点击待办事项的删除按钮
2. 确认删除操作

### 筛选待办事项

**按状态筛选**：

1. 点击筛选按钮
2. 在"状态"下拉菜单中选择状态
3. 系统显示符合条件的待办事项

**按优先级筛选**：

1. 点击筛选按钮
2. 在"优先级"下拉菜单中选择优先级
3. 系统显示符合条件的待办事项

**按分类筛选**：

1. 点击筛选按钮
2. 在"分类"下拉菜单中选择分类
3. 系统显示符合条件的待办事项

### 待办事项排序

**拖拽排序**：

1. 按住待办事项
2. 拖动到目标位置
3. 松开鼠标完成排序

### 批量操作

**清除已完成**：

1. 点击待办事项区域右上角的"更多"按钮
2. 选择"清除已完成"
3. 确认清除操作

**导出待办**：

1. 点击待办事项区域右上角的"更多"按钮
2. 选择"导出待办"
3. 系统导出待办事项为CSV文件

### 逾期提醒

系统会自动检查待办事项的截止日期，对于逾期的待办事项：

- 在待办事项列表中显示"已逾期"标记
- 在通知中心发送逾期提醒
- 逾期的待办事项会以红色高亮显示

## 🔔 通知中心

### 什么是通知中心

通知中心集中展示系统通知、待办事项、系统消息等，帮助您及时了解重要信息。

### 查看通知

**打开通知面板**：

1. 点击顶部导航栏的通知铃铛图标
2. 通知面板会展开显示所有通知

**通知类型**：

- 全部：显示所有通知
- 待办：显示待办事项通知
- 系统：显示系统消息
- 订单：显示订单相关通知
- 警告：显示警告信息

### 通知操作

**标记已读**：

1. 点击通知卡片
2. 通知标记为已读
3. 已读的通知会变灰显示

**全部已读**：

1. 点击通知面板右上角的"全部已读"按钮
2. 所有未读通知标记为已读

**删除通知**：

1. 点击通知卡片右上角的删除按钮
2. 通知从列表中删除

**通知操作**：

部分通知包含操作按钮，点击按钮可以执行相应操作。

### 通知设置

**打开设置**：

1. 点击通知面板右上角的设置图标
2. 在弹出的对话框中进行设置

**通知类型设置**：

- 选择要接收的通知类型
- 设置通知优先级

**通知方式设置**：

- 浏览器通知：启用/禁用浏览器通知
- 提示音效：启用/禁用提示音效
- 提示音量：调整提示音量

**通知时间设置**：

- 设置通知显示时间
- 设置通知自动标记已读时间

## ❓ 常见问题

### 数据相关问题

**Q：为什么数据没有实时更新？**

A：请检查以下几点：
1. 确认网络连接正常
2. 检查WebSocket连接状态
3. 刷新页面重新加载数据

**Q：如何导出数据报表？**

A：
1. 在数据看板中点击"导出报表"按钮
2. 选择导出格式（Excel/PDF）
3. 选择时间范围
4. 点击"确定"开始导出

**Q：数据统计不准确怎么办？**

A：
1. 检查时间范围设置是否正确
2. 联系技术支持进行数据校验
3. 确认数据源是否正常

### 快捷操作相关问题

**Q：如何添加自定义快捷操作？**

A：
1. 点击"添加快捷操作"按钮
2. 填写操作信息
3. 点击"确定"保存

**Q：快捷操作无法点击怎么办？**

A：
1. 检查是否有权限访问该功能
2. 确认跳转路径是否正确
3. 联系管理员检查权限设置

**Q：如何恢复默认快捷操作？**

A：
1. 右键点击快捷操作
2. 选择"重置位置"
3. 快捷操作恢复到默认位置

### 待办事项相关问题

**Q：待办事项提醒不工作怎么办？**

A：
1. 检查浏览器通知权限
2. 确认提醒功能已启用
3. 检查系统时间设置

**Q：如何批量删除待办事项？**

A：
1. 点击"更多"按钮
2. 选择"清除已完成"
3. 确认清除操作

**Q：待办事项可以同步到其他设备吗？**

A：是的，待办事项会自动同步到所有登录的设备。

### 通知相关问题

**Q：为什么收不到通知？**

A：请检查以下几点：
1. 确认浏览器通知权限已开启
2. 检查通知设置是否正确
3. 确认网络连接正常

**Q：如何关闭特定类型的通知？**

A：
1. 打开通知中心
2. 点击设置图标
3. 在通知类型设置中关闭不需要的通知

**Q：通知太多怎么办？**

A：
1. 点击"全部已读"快速标记所有通知
2. 调整通知设置，减少通知频率
3. 设置通知优先级，只接收重要通知

### 其他问题

**Q：系统运行缓慢怎么办？**

A：
1. 清除浏览器缓存
2. 关闭不必要的浏览器标签
3. 检查网络连接速度
4. 联系技术支持
