# 菜单管理功能模块用户手册

## 模块概述

菜单管理模块是YYC³餐饮行业智能化平台的核心功能模块，提供菜品管理、分类管理、权限控制、批量操作等功能，帮助管理者高效管理餐厅菜单。

## 功能导航

### 1. 菜品管理

#### 1.1 功能说明
菜品管理模块提供菜品的添加、编辑、删除、查询等功能，支持菜品信息的完整管理。

#### 1.2 菜品信息
- 基本信息：菜品名称、分类、价格
- 详细信息：描述、制作时间、图片
- 状态管理：上架、下架
- 统计信息：创建时间、更新时间

#### 1.3 菜品操作
**添加菜品**：
1. 点击"添加菜品"按钮
2. 填写菜品基本信息
3. 上传菜品图片
4. 设置价格和描述
5. 点击"创建"按钮

**编辑菜品**：
1. 找到要编辑的菜品
2. 点击"编辑"按钮
3. 修改菜品信息
4. 点击"更新"按钮

**删除菜品**：
1. 找到要删除的菜品
2. 点击"删除"按钮
3. 在确认对话框中点击"确定"

**上下架菜品**：
1. 找到要操作的菜品
2. 点击状态开关
3. 系统自动更新状态

#### 1.4 菜品筛选
- 按分类筛选：主食、汤品、小菜、饮料、甜点
- 按状态筛选：上架、下架
- 关键词搜索：菜品名称

### 2. 分类管理

#### 2.1 功能说明
分类管理模块提供菜单分类的层级管理功能，支持多级分类结构。

#### 2.2 分类结构
- 一级分类：主食、汤品、小菜、饮料、甜点
- 二级分类：米饭类、面食类、清汤、浓汤等
- 支持无限层级扩展

#### 2.3 分类操作
**添加分类**：
1. 点击"添加分类"按钮
2. 填写分类名称
3. 选择父分类（可选）
4. 选择图标
5. 设置排序值
6. 点击"创建"按钮

**编辑分类**：
1. 在分类树中找到要编辑的分类
2. 点击分类的编辑按钮
3. 修改分类信息
4. 点击"更新"按钮

**添加子分类**：
1. 在分类树中找到父分类
2. 点击父分类的添加按钮
3. 填写子分类信息
4. 点击"创建"按钮

**删除分类**：
1. 在分类树中找到要删除的分类
2. 点击分类的删除按钮
3. 在确认对话框中点击"确定"
4. 注意：有菜品的分类无法删除

**启用/禁用分类**：
1. 选择要操作的分类
2. 点击"启用"或"禁用"按钮
3. 系统自动更新分类状态

#### 2.4 分类图标
系统提供多种图标供选择：
- 餐具（Dish）
- 汉堡（Hamburger）
- 咖啡（Coffee）
- 饮料（Goblet）
- 甜点（IceCream）
- 水果（Cherry）
- 蔬菜（Vegetables）
- 肉类（Chicken）

### 3. 权限管理

#### 3.1 功能说明
权限管理模块提供菜单访问权限的精细化控制，支持基于角色的权限分配。

#### 3.2 角色类型
- 超级管理员：拥有所有权限
- 店长：管理店铺日常运营
- 厨师长：管理厨房菜品
- 服务员：处理订单和客户服务
- 收银员：处理收款和账单

#### 3.3 权限分组
**菜单管理**：
- 查看菜单（只读）
- 创建菜品（读写）
- 编辑菜品（读写）
- 删除菜品（读写）
- 导出菜单（只读）

**分类管理**：
- 查看分类（只读）
- 创建分类（读写）
- 编辑分类（读写）
- 删除分类（读写）

**价格管理**：
- 查看价格（只读）
- 修改价格（读写）
- 审批价格（读写）

**库存管理**：
- 查看库存（只读）
- 更新库存（读写）
- 库存预警（只读）

#### 3.4 权限配置
**配置角色权限**：
1. 在角色列表中选择角色
2. 在权限配置区域勾选相应权限
3. 点击"保存配置"按钮
4. 系统自动应用权限设置

**添加权限规则**：
1. 点击"添加权限规则"按钮
2. 选择角色和菜单分类
3. 设置权限级别（只读/读写/完全控制）
4. 选择操作权限
5. 点击"创建"按钮

#### 3.5 权限级别
- 只读：只能查看数据，不能修改
- 读写：可以查看和修改数据
- 完全控制：拥有所有操作权限

### 4. 批量操作

#### 4.1 功能说明
批量操作模块提供对多个菜品的同时操作功能，提高管理效率。

#### 4.2 批量选择
- 在菜品列表中勾选多个菜品
- 系统显示已选择菜品数量和统计信息
- 点击"清空选择"可以取消所有选择

#### 4.3 批量状态更新
**批量上架**：
1. 选择要上架的菜品
2. 展开"状态更新"操作组
3. 点击"批量上架"按钮
4. 在确认对话框中点击"确定"

**批量下架**：
1. 选择要下架的菜品
2. 展开"状态更新"操作组
3. 点击"批量下架"按钮
4. 在确认对话框中点击"确定"

#### 4.4 批量价格调整
**固定价格调整**：
1. 选择要调整价格的菜品
2. 展开"价格调整"操作组
3. 选择"固定价格"模式
4. 输入新的价格
5. 点击"应用价格调整"按钮
6. 在预览对话框中查看变化
7. 点击"确认操作"按钮

**百分比调整**：
1. 选择要调整价格的菜品
2. 展开"价格调整"操作组
3. 选择"百分比调整"模式
4. 输入调整百分比（正数为涨价，负数为降价）
5. 点击"应用价格调整"按钮
6. 在预览对话框中查看变化
7. 点击"确认操作"按钮

#### 4.5 批量分类调整
1. 选择要调整分类的菜品
2. 展开"分类调整"操作组
3. 选择目标分类
4. 点击"更新分类"按钮
5. 在确认对话框中点击"确定"

#### 4.6 批量导出
**导出Excel**：
1. 选择要导出的菜品
2. 展开"导出操作"操作组
3. 点击"导出Excel"按钮
4. 系统自动下载Excel文件

**导出PDF**：
1. 选择要导出的菜品
2. 展开"导出操作"操作组
3. 点击"导出PDF"按钮
4. 系统自动下载PDF文件

**导出CSV**：
1. 选择要导出的菜品
2. 展开"导出操作"操作组
3. 点击"导出CSV"按钮
4. 系统自动下载CSV文件

#### 4.7 批量打印
**打印菜单**：
1. 选择要打印的菜品
2. 展开"打印操作"操作组
3. 点击"打印菜单"按钮
4. 系统在新窗口中打开打印预览

**打印二维码**：
1. 选择要打印二维码的菜品
2. 展开"打印操作"操作组
3. 点击"打印二维码"按钮
4. 系统在新窗口中显示二维码

#### 4.8 批量删除
1. 选择要删除的菜品
2. 展开"删除操作"操作组
3. 点击"批量删除"按钮
4. 在确认对话框中点击"确定"
5. 注意：此操作不可恢复

### 5. 菜单预览

#### 5.1 功能说明
菜单预览模块提供菜单的实时预览功能，方便查看菜单在客户端的显示效果。

#### 5.2 预览模式
- 列表视图：以列表形式展示菜品
- 网格视图：以网格形式展示菜品
- 分类视图：按分类展示菜品

#### 5.3 预览操作
**查看预览**：
1. 点击"预览"按钮
2. 选择预览模式
3. 系统显示菜单预览

**分享预览**：
1. 在预览页面点击"分享"按钮
2. 复制分享链接
3. 发送给客户或团队成员

### 6. 菜单统计

#### 6.1 功能说明
菜单统计模块提供菜单数据的统计分析功能，帮助了解菜品销售情况。

#### 6.2 统计指标
- 菜品总数
- 上架菜品数
- 下架菜品数
- 分类数量
- 平均价格
- 最高价格
- 最低价格

#### 6.3 销售统计
- 销量排行
- 销售额排行
- 分类销售占比
- 时间段销售趋势

### 7. 菜单审核

#### 7.1 功能说明
菜单审核模块提供菜品变更的审核流程，确保菜单信息的准确性。

#### 7.2 审核流程
**提交审核**：
1. 修改菜品信息
2. 点击"提交审核"按钮
3. 系统发送审核通知

**审核菜品**：
1. 进入审核列表
2. 查看菜品变更详情
3. 选择审核结果（通过/拒绝）
4. 填写审核意见
5. 点击"提交审核"按钮

**审核历史**：
1. 查看所有审核记录
2. 查看审核详情
3. 查看审核意见

## 常见问题

### Q1: 如何快速批量调整价格？
A: 使用批量操作功能，选择要调整价格的菜品，选择价格调整模式（固定或百分比），输入调整值，系统会显示预览，确认后即可批量调整。

### Q2: 如何设置菜品分类的层级？
A: 在分类管理中，可以添加子分类。创建分类时选择父分类，系统会自动建立层级关系。

### Q3: 如何控制不同角色的菜单访问权限？
A: 在权限管理中，选择角色，勾选相应的权限，保存配置即可。系统会根据权限控制角色的访问和操作。

### Q4: 如何导出菜单数据？
A: 选择要导出的菜品，在批量操作中选择导出格式（Excel/PDF/CSV），系统会自动下载文件。

### Q5: 如何打印菜单二维码？
A: 选择要打印二维码的菜品，在批量操作中点击"打印二维码"，系统会生成并显示二维码。

### Q6: 如何禁用有菜品的分类？
A: 首先将分类下的菜品移动到其他分类或下架，然后才能禁用或删除该分类。

### Q7: 如何查看菜单的销售统计？
A: 在菜单统计模块中，可以查看菜品的销量排行、销售额排行、分类销售占比等统计数据。

### Q8: 如何提交菜品变更审核？
A: 修改菜品信息后，点击"提交审核"按钮，系统会发送审核通知给审核人员。
