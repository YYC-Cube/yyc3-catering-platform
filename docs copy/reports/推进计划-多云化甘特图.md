```
开发 → 测试 → 预生产 → 生产
 ↓      ↓       ↓       ↓
Dev    Staging  Staging  Prod
```

### 7.3 发布流程

1. **创建发布分支**
   ```bash
   git checkout -b release/v2.0.0
   ```

2. **测试验证**
   ```bash
   pnpm test
   pnpm lint
   pnpm type-check
   ```

3. **构建镜像**
   ```bash
   docker build -t yyc3/service:2.0.0 .
   ```

4. **推送镜像**
   ```bash
   docker push yyc3/service:2.0.0
   ```

5. **Helm部署**
   ```bash
   helm upgrade yyc3 ./helm -f values-prod.yaml
   ```

6. **验证发布**
   ```bash
   kubectl rollout status deployment/service
   ```
