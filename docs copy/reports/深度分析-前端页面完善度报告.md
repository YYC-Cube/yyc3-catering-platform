### 5.2 性能优化建议

#### 5.2.1 首屏加载优化

**问题**：首屏加载时间> 3s

**解决方案**：
1. **关键路径CSS内联**
   ```html
   <style>
     /* 关键CSS */
     .header { ... }
     .hero { ... }
   </style>
   ```

2. **图片懒加载**
   ```html
   <img src="placeholder.jpg" data-src="actual.jpg" loading="lazy">
   ```

3. **代码分割**
   ```js
   const Home = lazy(() => import('./Home'));
   ```

4. **预加载关键资源**
   ```html
   <link rel="preload" href="critical.js" as="script">
   ```

**预期效果**：首屏加载时间从4s降至2s

---

#### 5.2.2 图片优化

**问题**：图片体积大，加载慢

**解决方案**：
1. **WebP格式**
   ```js
   // 自动转换为WebP
   <picture>
     <source srcset="image.webp" type="image/webp">
     <img src="image.jpg" alt="...">
   </picture>
   ```

2. **响应式图片**
   ```js
   <img srcset="small.jpg 500w, medium.jpg 1000w"
        sizes="(max-width: 500px) 500px, 1000px">
   ```

3. **图片压缩**
   ```js
   // 使用Sharp压缩
   const image = sharp('original.jpg')
     .resize(800, 600)
     .webp({ quality: 80 });
   ```

4. **CDN加速**
   ```yaml
   cdn:
     provider: cloudfront
     origin: s3://yyc3-assets
     caching:
       ttl: 86400
   ```

**预期效果**：图片加载时间减少70%

---

#### 5.2.3 列表优化

**问题**：大列表性能差，滚动卡顿

**解决方案**：
1. **虚拟滚动**
   ```js
   // 使用react-window
   import { FixedSizeList as List } from 'react-window';

   <List height={500} itemCount={10000} itemSize={50}>
     {({ index, style }) => <div style={style}>Item {index}</div>}
   </List>
   ```

2. **分页加载**
   ```js
   // 无限滚动
   const loadMore = () => {
     setPage(prev => prev + 1);
   };
   ```

3. **数据缓存**
   ```js
   // 使用SWR或React Query
   const { data } = useSWR('/api/orders', fetcher);
   ```

**预期效果**：10000条列表滚动流畅

### 5.3 安全优化建议

#### 5.3.1 XSS防护

**问题**：前端存在XSS风险

**解决方案**：
1. **输入过滤**
   ```js
   const sanitize = (input) => {
     return input.replace(/<[^>]*>/g, '');
   };
   ```

2. **输出转义**
   ```js
   // 使用DOMPurify
   const clean = DOMPurify.sanitize(dirty);
   ```

3. **CSP策略**
   ```http
   Content-Security-Policy: default-src 'self'; script-src 'self' 'unsafe-inline'
   ```

#### 5.3.2 CSRF防护

**问题**：CSRF攻击风险

**解决方案**：

   ```js
   // 后端生成Token
   const csrfToken = generateToken();

   // 前端发送Token

     'X-CSRF-Token': csrfToken
   }
   ```

2. **SameSite Cookie**
   ```http
   Set-Cookie: sessionid=...; SameSite=Strict
   ```
