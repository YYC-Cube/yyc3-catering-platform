## 1. 平台概述

### 1.1 平台简介

YYC³餐饮行业智能化平台是一款专为餐饮企业打造的全方位智能化管理平台，集成了客户管理、订单管理、厨房管理、菜单管理、数据分析等核心功能，帮助餐饮企业实现数字化转型，提升运营效率，优化客户体验。

### 1.2 核心功能

- **客户管理**：全方位客户信息管理，支持客户分群、标签管理、生命周期跟踪
- **订单管理**：订单全流程管理，支持订单查询、状态跟踪、统计分析
- **厨房管理**：厨房任务管理，支持任务分配、进度跟踪、效率分析
- **菜单管理**：菜品信息管理，支持分类管理、价格管理、库存管理
- **数据分析**：多维度数据分析，支持销售分析、客户分析、趋势分析
- **系统管理**：系统配置管理，支持用户管理、权限管理、日志管理

### 1.3 系统要求

- **浏览器**：Chrome 80+、Firefox 75+、Safari 13+、Edge 80+
- **分辨率**：建议使用1920x1080或更高分辨率
- **网络**：稳定的互联网连接

## 2. 快速入门

### 2.1 登录系统

1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击"登录"按钮
4. 首次登录需要修改密码

### 2.2 界面导航

系统采用左侧导航栏+顶部工具栏的布局：

- **左侧导航栏**：主要功能模块入口
- **顶部工具栏**：用户信息、消息通知、系统设置
- **主内容区**：当前功能模块的操作界面

### 2.3 基本操作

#### 2.3.1 表格操作

- **查看详情**：点击操作列的"查看"按钮

## 3. 客户管理模块

### 3.1 客户列表

#### 3.1.1 功能概述

客户列表展示所有客户的基本信息，支持搜索、筛选、排序、批量操作等功能。

#### 3.1.2 操作步骤

1. 进入"客户管理"→"客户列表"
2. 查看客户列表，包含以下信息：
   - 客户姓名
   - 手机号
   - 邮箱
   - 会员等级
   - 累计消费
   - 订单数
   - 积分
   - 注册时间
3. 使用搜索框或筛选条件查找客户
4. 点击操作按钮进行相应操作

#### 3.1.3 搜索筛选

支持以下筛选条件：
- **客户姓名**：输入客户姓名进行模糊搜索
- **手机号**：输入手机号进行精确搜索
- **会员等级**：选择会员等级进行筛选
- **注册时间**：选择注册时间范围
- **状态**：选择客户状态（活跃/非活跃/黑名单）

### 3.2 客户详情

#### 3.2.1 功能概述

客户详情展示客户的完整信息，包括基本信息、订单历史、积分历史、状态历史等。

#### 3.2.2 操作步骤

1. 在客户列表中点击"查看详情"按钮
2. 查看客户基本信息
3. 切换标签页查看不同类型的历史记录：
   - 订单历史
   - 积分历史
   - 状态历史
   - 访问记录

#### 3.2.3 信息说明

**基本信息**：
- 客户姓名、手机号、邮箱
- 会员等级、累计消费、订单数、积分
- 注册时间、最后访问时间

**订单历史**：
- 订单号、订单金额、订单状态
- 支付方式、下单时间

**积分历史**：
- 积分变化、变化后积分
- 变化类型（获得/消费/过期）
- 说明、时间

**状态历史**：
- 旧状态、新状态
- 变更人、变更时间、变更原因

### 3.3 客户分群

#### 3.3.1 功能概述

客户分群支持将客户按照不同维度进行分组管理，便于精细化运营。

#### 3.3.3 分群类型说明

- **手动分群**：手动添加客户到分群
- **自动分群**：根据预设条件自动添加客户
- **RFM分群**：根据RFM模型自动分群
- **生命周期分群**：根据客户生命周期阶段自动分群

#### 3.3.4 管理分群成员

1. 在分群列表中点击"查看成员"
2. 查看分群成员列表
3. 点击"添加成员"添加客户

### 3.6 客户生命周期管理

#### 3.6.1 功能概述

客户生命周期管理基于RFM模型和规则引擎，自动管理客户生命周期阶段，提供流失预警、智能关怀等功能，帮助提升客户留存率和客户价值。

#### 3.6.2 生命周期阶段

系统定义了6个生命周期阶段：

1. **潜在客户**：已注册但未下单
2. **新客户**：首次下单后30天内
3. **活跃客户**：30天内有消费记录
4. **忠诚客户**：高价值、高频次消费
5. **休眠客户**：60-90天未消费
6. **流失客户**：90天以上未消费

#### 3.6.3 查看生命周期看板

1. 进入"客户管理"→"生命周期管理"
2. 查看生命周期数据看板
3. 包含以下内容：
   - 生命周期阶段分布
   - 客户价值分布（RFM）
   - 生命周期流转趋势
   - 客户留存率趋势
   - 客户生命周期价值分布

#### 3.6.4 RFM分析

##### 3.6.4.1 RFM模型说明

RFM模型通过三个维度评估客户价值：

- **Recency（最近消费）**：客户最近一次消费时间，分值1-5分
- **Frequency（消费频率）**：客户消费频次，分值1-5分
- **Monetary（消费金额）**：客户累计消费金额，分值1-5分

RFM总分 = Recency + Frequency + Monetary（3-15分）

##### 3.6.4.2 客户等级划分

| RFM分数 | 客户等级 | 说明 |
|---------|----------|------|
| 13-15分 | 超级VIP | 最高价值客户，需要重点维护 |
| 10-12分 | VIP | 高价值客户，需要持续关注 |
| 7-9分 | 会员 | 中等价值客户，需要提升价值 |
| 4-6分 | 普通 | 一般价值客户，需要培养 |
| 1-3分 | 低价值 | 低价值客户，需要激活 |

##### 3.6.4.3 计算RFM评分

1. 进入"客户管理"→"RFM分析"
2. 点击"计算RFM评分"按钮
3. 选择要计算的客户（支持批量选择）
4. 点击"确定"开始计算
5. 系统自动计算并更新RFM评分

##### 3.6.4.4 查看RFM分析报告

1. 在RFM分析页面查看统计数据
2. 包含以下内容：
   - 平均RFM分数
   - 客户等级分布
   - Recency分布
   - Frequency分布
   - Monetary分布
   - RFM三维分布图

##### 3.6.4.5 导出RFM分析报告

1. 在RFM分析页面点击"导出报告"
2. 选择导出格式（Excel/PDF）
3. 选择时间范围
4. 点击"确定"导出

#### 3.6.5 客户流失预警

##### 3.6.5.1 流失预警说明

系统根据客户行为数据预测客户流失概率，并生成流失预警：

- **高危预警**：流失概率≥70%，需要立即处理
- **中危预警**：流失概率40%-70%，需要关注
- **低危预警**：流失概率<40%，需要监控

##### 3.6.5.2 查看流失预警列表

1. 进入"客户管理"→"流失预警"
2. 查看流失预警列表
3. 包含以下信息：
   - 客户姓名、手机号、邮箱
   - 流失概率、风险等级
   - 风险因素
   - 预测流失日期
   - 推荐挽回措施
   - 负责人、状态

##### 3.6.5.3 筛选流失预警

支持以下筛选条件：
- **风险等级**：高危/中危/低危
- **状态**：待处理/处理中/已解决/已关闭
- **负责人**：选择负责人
- **时间范围**：选择预警时间范围

##### 3.6.5.4 分配流失预警

1. 在流失预警列表中找到待分配的预警
2. 点击"分配"按钮
3. 选择负责人
4. 点击"确定"保存
5. 预警状态变为"处理中"

##### 3.6.5.5 处理流失预警

1. 在流失预警列表中点击"查看详情"
2. 查看客户信息和风险因素
3. 根据推荐挽回措施制定处理方案
4. 记录处理过程和结果
5. 点击"标记为已解决"完成处理

##### 3.6.5.6 查看流失趋势

1. 在流失预警页面点击"趋势分析"
2. 查看流失趋势图表
3. 包含以下内容：
   - 流失客户数量趋势
   - 流失率趋势
   - 各风险等级预警数量趋势
   - 流失原因分析

#### 3.6.6 智能客户关怀

##### 3.6.6.1 关怀提醒说明

系统根据客户行为和生命周期阶段，自动生成关怀提醒：

- **回访提醒**：客户长时间未消费时触发
- **生日提醒**：客户生日当天触发
- **节日提醒**：重要节日前触发
- **会员升级提醒**：客户等级提升时触发
- **消费提醒**：客户消费后触发
