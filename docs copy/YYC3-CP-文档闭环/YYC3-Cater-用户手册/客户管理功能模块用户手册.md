# 客户管理功能模块用户手册

## 模块概述

客户管理模块是YYC³餐饮行业智能化平台的核心功能模块，提供客户信息的创建、查询、更新、删除(CRUD)基础功能，以及客户分类管理、客户状态跟踪、客户历史记录查询等扩展功能，帮助管理者全面了解和管理客户关系。

## 功能导航

### 1. 客户信息管理

#### 1.1 功能说明
客户信息管理模块提供客户基本信息的完整管理功能，包括客户档案、联系方式、消费记录等。

#### 1.2 客户信息字段
- 基本信息：客户ID、姓名、性别、生日
- 联系方式：手机号码、电子邮箱、地址
- 会员信息：会员等级、积分余额
- 消费统计：订单数量、消费金额、客单价
- 状态信息：客户状态（活跃/休眠/黑名单）
- 时间信息：注册时间、最后访问时间

#### 1.3 客户操作
**新增客户**：
1. 点击"新增客户"按钮
2. 填写客户基本信息
3. 设置联系方式
4. 选择会员等级
5. 点击"保存"按钮

**编辑客户**：
1. 找到要编辑的客户
2. 点击"编辑"按钮
3. 修改客户信息
4. 点击"保存"按钮

**删除客户**：
1. 找到要删除的客户
2. 点击"删除"按钮
3. 在确认对话框中点击"确定"
4. 注意：删除操作不可恢复

**查看详情**：
1. 找到要查看的客户
2. 点击"详情"按钮
3. 查看客户完整信息
4. 可切换不同标签页查看详细信息

#### 1.4 客户筛选
- 关键词搜索：客户姓名、手机号、邮箱
- 客户等级筛选：普通客户、会员、VIP、新客户
- 客户分群筛选：高价值客户、高频客户、新客户、流失风险客户、忠诚客户、休眠客户
- 时间范围筛选：注册时间、最后访问时间
- 状态筛选：活跃、休眠、黑名单

### 2. 客户状态跟踪

#### 2.1 功能说明
客户状态跟踪模块提供客户状态的实时监控和历史记录功能，帮助管理者了解客户的生命周期状态。

#### 2.2 客户状态类型
- **活跃**：客户近期有消费行为
- **休眠**：客户超过一定时间未访问
- **流失**：客户长时间未访问，可能已流失
- **黑名单**：客户被列入黑名单

#### 2.3 状态跟踪功能
**查看当前状态**：
1. 进入客户详情页面
2. 切换到"状态跟踪"标签页
3. 查看当前状态和状态描述
4. 查看状态变化历史时间线

**添加状态记录**：
1. 点击"添加状态"按钮
2. 选择状态类型
3. 填写备注信息
4. 填写操作人姓名
5. 点击"添加"按钮

**刷新状态**：
1. 点击"刷新"按钮
2. 系统自动更新状态数据
3. 查看最新的状态信息

#### 2.4 状态指标
- 活跃天数：客户活跃的总天数
- 总订单数：客户累计订单数量
- 平均间隔：两次消费之间的平均时间间隔
- 忠诚度：基于消费行为计算的忠诚度评分

#### 2.5 状态预测
系统基于客户行为分析，提供状态预测功能：
- 预测客户未来的状态变化
- 显示预测的概率和原因
- 帮助管理者提前采取措施

### 4. 客户分群管理

#### 4.1 功能说明
客户分群管理模块提供客户分群的创建、编辑、删除功能，支持手动分群、自动分群和AI智能分群。

#### 4.2 分群类型
- **手动分群**：由管理员手动创建和管理
- **自动分群**：基于预设规则自动生成
- **AI智能分群**：由AI算法智能识别和生成

#### 4.3 预设分群
- 高价值客户：消费金额高、订单频次高的优质客户
- 高频客户：订单频次高，但单次消费金额适中的客户
- 新客户：最近30天内注册的新客户
- 流失风险客户：AI识别出的有流失风险的客户
- 休眠客户：超过90天未访问的客户
- 忠诚客户：长期活跃且消费稳定的客户

#### 4.4 分群操作
**查看分群**：
1. 进入客户分群管理页面
2. 在左侧列表中选择分群
3. 查看分群详细信息
4. 查看分群规则和客户统计

**新建分群**：
1. 点击"新建分群"按钮
2. 填写分群名称
3. 选择分群类型
4. 填写分群描述
5. 设置分群规则
6. 选择逻辑关系（满足所有规则/满足任一规则）
7. 点击"创建"按钮

**编辑分群**：
1. 在左侧列表中选择要编辑的分群
2. 点击"编辑分群"按钮
3. 修改分群信息
4. 点击"更新"按钮

**删除分群**：
1. 在左侧列表中选择要删除的分群
2. 点击删除按钮
3. 在确认对话框中点击"确定"

#### 4.5 分群规则
**规则类型**：
- 消费金额：基于客户累计消费金额
- 订单数量：基于客户累计订单数量
- 最后访问：基于客户最后访问时间
- 会员等级：基于客户会员等级
- 注册时间：基于客户注册时间

**操作符**：
- 大于（>）：大于指定值
- 小于（<）：小于指定值
- 等于（=）：等于指定值
- 包含：包含指定内容

**规则组合**：
- 满足所有规则（AND）：客户必须满足所有规则条件
- 满足任一规则（OR）：客户只需满足任一规则条件

#### 4.6 分群统计
- 客户数：分群包含的客户数量
- 占比：分群客户占总客户数的百分比
- 平均消费：分群客户的平均消费金额
- 平均订单：分群客户的平均订单数量

#### 4.7 分群导出
1. 选择要导出的分群
2. 点击"导出客户"按钮
3. 系统自动下载Excel文件

#### 4.8 分群分析
1. 选择要分析的分群
2. 点击"分析报告"按钮

4. 了解分群的特征和趋势

### 8. 生命周期管理

#### 8.1 功能说明
生命周期管理模块提供客户生命周期的全面跟踪和管理功能，帮助管理者了解客户从新客户到忠诚客户的完整生命周期过程。

#### 8.2 生命周期阶段
系统将客户生命周期划分为以下阶段：

- **新客户**：注册时间不超过30天
- **活跃客户**：最近30天内有消费记录
- **沉睡客户**：30-90天无消费记录
- **流失客户**：超过90天无消费记录
- **忠诚客户**：消费频次和金额达到一定标准

#### 8.3 生命周期阶段特征

**新客户特征**：
- 注册时间短，消费频次低
- 对平台了解不深
- 需要引导和关怀
- 容易流失

**活跃客户特征**：
- 近期有消费行为
- 消费频次较高
- 对平台有一定了解
- 需要保持活跃度

**沉睡客户特征**：
- 30-90天无消费记录
- 消费频次下降
- 可能对服务不满意
- 有流失风险

**流失客户特征**：
- 超过90天无消费记录
- 消费频次为零
- 可能已转向竞争对手
- 需要挽回措施

**忠诚客户特征**：
- 消费频次高
- 消费金额大
- 对平台忠诚度高
- 是核心客户群体

#### 8.4 管理客户生命周期

4. 了解客户生命周期整体情况

**查看阶段客户列表**：

2. 查看该阶段的所有客户
3. 查看客户详细信息
4. 对客户进行操作

**调整客户生命周期阶段**：
1. 选择需要调整的客户
2. 点击"调整阶段"按钮
3. 选择目标生命周期阶段
4. 填写调整原因
5. 点击"确认"完成调整

**批量调整生命周期阶段**：
1. 勾选多个客户
2. 点击"批量调整阶段"按钮
3. 选择目标生命周期阶段
4. 填写调整原因
5. 点击"确认"完成批量调整

#### 8.5 生命周期操作

### 9. 客户价值评估（RFM模型）

#### 9.1 功能说明
客户价值评估功能基于RFM模型（最近消费、消费频次、消费金额）对客户进行价值评估和分群，帮助管理者识别高价值客户，制定差异化营销策略。

#### 9.2 RFM模型说明

**R（Recency）- 最近消费**：
- 定义：客户最近一次消费的时间
- 作用：反映客户活跃度
- 评分：最近消费时间越短，评分越高

**F（Frequency）- 消费频次**：
- 定义：客户在指定时间段内的消费次数
- 作用：反映客户忠诚度
- 评分：消费频次越高，评分越高

**M（Monetary）- 消费金额**：
- 定义：客户在指定时间段内的累计消费金额
- 作用：反映客户价值
- 评分：消费金额越高，评分越高

#### 9.3 RFM评分规则

**R评分规则**：
- 5分：最近消费≤7天
- 4分：7天<最近消费≤15天
- 3分：15天<最近消费≤30天
- 2分：30天<最近消费≤60天
- 1分：最近消费>60天

**F评分规则**：
- 5分：消费频次≥20次
- 4分：15次≤消费频次<20次
- 3分：10次≤消费频次<15次
- 2分：5次≤消费频次<10次
- 1分：消费频次<5次

**M评分规则**：
- 5分：消费金额≥5000元
- 4分：3000元≤消费金额<5000元
- 3分：1000元≤消费金额<3000元
- 2分：500元≤消费金额<1000元
- 1分：消费金额<500元

## 常见问题

### Q1: 如何快速查找特定客户？
A: 使用搜索功能，输入客户姓名、手机号或邮箱进行搜索。也可以使用筛选功能，按客户等级、分群、状态等条件进行筛选。

### Q2: 如何将客户加入黑名单？
A: 在客户列表中找到要操作的客户，点击"更多"按钮，选择"加入黑名单"，在确认对话框中点击"确定"。

### Q3: 如何查看客户的完整历史记录？
A: 进入客户详情页面，切换到"历史记录"标签页，可以查看订单、积分、优惠券、评价、投诉等各类历史记录。

### Q4: 如何创建自定义客户分群？
A: 进入客户分群管理页面，点击"新建分群"，填写分群信息，设置分群规则，选择逻辑关系，点击"创建"按钮。

### Q5: 如何导出客户数据？
A: 在客户列表页面点击"导出客户"按钮，选择导出格式和字段，系统会自动下载文件。也可以先筛选或勾选客户，再导出。

### Q6: 如何导入客户数据？
A: 准备符合模板格式的Excel文件，点击"导入客户"按钮，选择文件，系统会自动验证并导入数据。

### Q7: 客户状态预测准确吗？
A: 系统基于客户行为数据进行AI分析，预测结果仅供参考。建议结合实际情况进行判断和决策。

### Q8: 如何调整客户积分？
A: 进入客户详情页面，切换到"会员管理"标签页，可以查看积分记录，也可以手动调整客户积分。
