# 订单管理功能模块用户手册

## 模块概述

订单管理模块是YYC³餐饮行业智能化平台的核心业务模块，提供订单创建、查询、更新、批量操作等全方位的订单管理功能，帮助管理者高效处理日常订单业务。

## 功能导航

### 1. 订单概览

#### 1.1 功能说明
订单概览页面展示今日订单的关键指标，包括订单数量、营收、待处理订单数、服务客户数等。

#### 1.2 指标说明
- **今日订单**: 当日创建的订单总数
- **今日营收**: 当日订单总金额
- **待处理**: 等待处理的订单数量
- **服务客户**: 当日服务的客户数量

#### 1.3 趋势显示
每个指标卡片下方显示趋势变化：
- 📈 绿色向上箭头：表示增长
- 📉 红色向下箭头：表示下降
- ➖ 灰色横线：表示稳定

### 2. 订单筛选

#### 2.1 筛选条件
- **订单状态**: 待确认、已确认、制作中、已完成、已上餐、已取消、已退款
- **订单类型**: 堂食、外带、外送
- **时间范围**: 自定义开始和结束时间
- **关键词搜索**: 订单号、客户姓名、电话号码

#### 2.2 使用方法
1. 在筛选工具栏中选择或输入筛选条件
2. 系统自动根据筛选条件更新订单列表
3. 可以组合多个筛选条件进行精确查询

### 3. 视图模式

#### 3.1 列表视图
**功能说明**：以表格形式展示订单列表，适合查看详细信息。

**特点**：
- 显示订单完整信息
- 支持排序和分页
- 可批量选择订单
- 每行显示订单号、客户信息、类型、状态、金额等

**使用方法**：
1. 点击"列表视图"按钮切换到列表模式
2. 使用表格上方的筛选条件过滤订单
3. 点击表头可按该列排序
4. 使用底部分页器翻页

#### 3.2 看板视图
**功能说明**：以看板形式展示订单，按状态分组，适合快速处理订单。

**特点**：
- 按订单状态分组显示
- 支持拖拽更新订单状态
- 直观展示各状态订单数量
- 卡片式展示订单信息

**使用方法**：
1. 点击"看板视图"按钮切换到看板模式
2. 拖拽订单卡片到不同状态列
3. 点击订单卡片查看详情
4. 使用卡片上的快捷按钮操作订单

#### 3.3 时间线视图
**功能说明**：以时间线形式展示订单，按创建时间排序，适合查看订单流程。

**特点**：
- 按时间顺序展示订单
- 显示订单创建时间
- 展示订单简要信息
- 适合查看订单处理流程

**使用方法**：
1. 点击"时间线"按钮切换到时间线模式
2. 查看按时间排序的订单列表
3. 点击订单查看详情

### 4. 订单操作

#### 4.1 单个订单操作
**查看详情**：
1. 点击订单行的"详情"按钮
2. 在弹出的对话框中查看订单完整信息
3. 包括基本信息、订单项目、费用汇总、备注等

**确认订单**：
1. 找到待确认的订单
2. 点击"确认"按钮
3. 系统将订单状态更新为"已确认"

**取消订单**：
1. 找到需要取消的订单
2. 点击"取消"按钮
3. 在确认对话框中点击"确定"
4. 系统将订单状态更新为"已取消"

**打印订单**：
1. 点击订单行的"更多"按钮
2. 选择"打印"选项
3. 系统将订单发送到打印机

**查看流程图**：
1. 点击订单行的"更多"按钮
2. 选择"流程图"选项
3. 在弹出的对话框中查看订单流程可视化

**退款处理**：
1. 点击订单行的"更多"按钮
2. 选择"退款"选项
3. 在确认对话框中点击"确定"
4. 系统将订单状态更新为"已退款"

#### 4.2 批量操作
**批量选择**：
1. 在列表视图中勾选需要操作的订单
2. 或在看板视图中多选订单卡片

**批量确认**：
1. 选择多个待确认的订单
2. 点击"批量操作"按钮
3. 选择"确认订单"选项
4. 在确认对话框中点击"确定"

**批量取消**：
1. 选择多个需要取消的订单
2. 点击"批量操作"按钮
3. 选择"取消订单"选项
4. 在确认对话框中点击"确定"

**批量导出**：
1. 选择需要导出的订单
2. 点击"批量操作"按钮
3. 选择"导出订单"选项
4. 选择导出格式（Excel、PDF、CSV）
5. 系统自动下载导出文件

**批量打印**：
1. 选择需要打印的订单
2. 点击"批量操作"按钮
3. 选择"批量打印"选项
4. 选择打印类型（厨房单、客户单、配送单）
5. 系统将订单发送到打印机

### 5. 订单状态跟踪

#### 5.1 功能说明
订单状态跟踪功能提供订单从创建到完成的完整状态流转记录，帮助管理者了解订单处理进度。

#### 5.2 状态步骤
1. **待确认**: 订单已创建，等待确认
2. **已确认**: 订单已确认，开始准备
3. **制作中**: 正在制作订单
4. **已完成**: 订单制作完成
5. **已上餐**: 订单已送达客户

#### 5.3 使用方法
1. 点击订单行的"详情"按钮
2. 在订单详情页面查看状态跟踪
3. 查看每个状态的时间戳和操作人
4. 查看状态转换耗时

#### 5.4 自动刷新
- 点击"自动"按钮开启自动刷新
- 系统每5秒自动更新订单状态
- 再次点击"自动"按钮停止自动刷新

### 6. 新建订单

#### 6.1 功能说明
支持手动创建新订单，适用于特殊情况下的订单录入。

#### 6.2 创建步骤
1. 点击"新建订单"按钮
2. 在弹出的对话框中填写订单信息：
   - 选择订单类型（堂食、外带、外送）
   - 选择客户信息
   - 添加订单项目
   - 填写备注信息
3. 点击"确定"按钮创建订单

#### 6.3 订单项目
- 点击"添加菜品"按钮
- 在菜品列表中选择菜品
- 设置菜品数量和规格
- 添加备注信息

### 7. 订单详情

#### 7.1 基本信息
- 订单号
- 桌号/配送地址
- 客户姓名
- 客户电话
- 下单时间
- 更新时间

#### 7.2 订单项目
- 菜品名称
- 分类
- 数量
- 单价
- 小计
- 备注

#### 7.3 费用汇总
- 菜品小计
- 服务费
- 优惠
- 总计

#### 7.4 订单备注
显示订单的特殊要求和备注信息。

### 8. 订单流程可视化

#### 8.1 功能说明
订单流程可视化以图形方式展示订单的完整处理流程，包括各阶段的时间、操作人等信息。

#### 8.2 使用方法
1. 点击订单行的"更多"按钮
2. 选择"流程图"选项
3. 在弹出的对话框中查看订单流程图
4. 流程图展示订单从创建到完成的完整路径

### 9. 高级筛选

#### 9.1 组合筛选
支持多个筛选条件的组合使用：
- 订单状态 + 订单类型
- 订单状态 + 时间范围
- 订单类型 + 时间范围 + 关键词
- 所有条件组合

#### 9.2 保存筛选
1. 设置好筛选条件
2. 点击"保存筛选"按钮
3. 输入筛选名称
4. 下次可直接使用保存的筛选条件

### 10. 订单统计

#### 10.1 实时统计
- 今日订单数
- 今日营收
- 待处理订单数
- 服务客户数

#### 10.2 历史统计
- 按日统计
- 按周统计
- 按月统计
- 自定义时间范围统计

## 常见问题

### Q1: 如何快速查找特定订单？
A: 使用关键词搜索功能，输入订单号、客户姓名或电话号码，系统会自动匹配相关订单。

### Q2: 如何批量处理订单？
A: 在列表视图中勾选多个订单，然后点击"批量操作"按钮，选择需要的操作类型。

### Q3: 订单状态如何流转？
A: 订单状态按以下顺序流转：待确认 → 已确认 → 制作中 → 已完成 → 已上餐。可以通过拖拽或操作按钮更新状态。

### Q4: 如何导出订单数据？
A: 选择需要导出的订单，点击"批量操作" → "导出订单"，选择导出格式（Excel、PDF、CSV）。

### Q5: 订单可以修改吗？
A: 已确认的订单可以修改，正在制作或已完成的订单需要取消后重新创建。

### Q6: 如何查看订单的处理历史？
A: 点击订单详情，在状态跟踪部分可以看到订单的完整处理历史，包括每个状态的时间戳和操作人。

### Q7: 订单可以打印哪些单据？
A: 支持打印厨房单、客户单、配送单三种类型。

### Q8: 如何处理退款订单？
A: 点击订单行的"更多"按钮，选择"退款"选项，在确认对话框中点击"确定"。
