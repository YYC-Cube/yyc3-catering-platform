# 厨房管理功能模块用户手册

## 模块概述

厨房管理模块是YYC³餐饮行业智能化平台的核心运营模块，提供厨房任务管理、员工分配、效率监控等功能，帮助管理者优化厨房运营效率。

## 功能导航

### 1. 厨房概览

#### 1.1 功能说明
厨房概览页面展示厨房运营的关键指标，包括进行中任务、今日完成、效率率、平均时间等。

#### 1.2 指标说明
- **进行中任务**: 当前正在处理的任务数量
- **今日完成**: 今日已完成的任务总数
- **效率率**: 任务完成效率百分比
- **平均时间**: 平均任务处理时间

#### 1.3 趋势显示
每个指标卡片下方显示趋势变化：
- 📈 绿色向上箭头：表示提升
- 📉 红色向下箭头：表示下降
- ➖ 灰色横线：表示稳定

### 2. 任务管理

#### 2.1 功能说明
任务管理模块提供厨房任务的创建、查询、更新、分配等功能。

#### 2.2 任务筛选
- **工作岗位**: 热菜、凉菜、汤类、吧台、点心、综合
- **任务状态**: 待处理、制作中、已完成、已上菜、已取消
- **优先级**: 低、普通、高、紧急
- **关键词搜索**: 菜品名称、订单号

#### 2.3 视图模式
**列表视图**：
- 以表格形式展示任务
- 显示任务完整信息
- 支持排序和分页
- 可批量选择任务

**看板视图**：
- 以看板形式展示任务
- 按状态分组显示
- 支持拖拽更新状态
- 直观展示任务分布

#### 2.4 任务操作
**查看详情**：
1. 点击任务行的"详情"按钮
2. 在弹出的对话框中查看任务完整信息

**开始任务**：
1. 找到待处理的任务
2. 点击"开始"按钮
3. 系统将任务状态更新为"制作中"

**完成任务**：
1. 找到制作中的任务
2. 点击"完成"按钮
3. 系统将任务状态更新为"已完成"

**分配任务**：
1. 点击任务行的"分配"按钮
2. 在弹出的对话框中选择员工
3. 设置优先级和备注
4. 点击"确定"按钮

### 3. 员工管理

#### 3.1 功能说明
员工管理模块提供厨房员工的添加、查询、编辑、删除等功能。

#### 3.2 员工信息
- 基本信息：姓名、工号、联系方式
- 岗位信息：岗位、技能等级
- 工作状态：在线、离线、忙碌
- 绩效统计：完成任务数、效率评分

#### 3.3 员工操作
**添加员工**：
1. 点击"添加员工"按钮
2. 填写员工基本信息
3. 设置岗位和技能
4. 点击"确定"按钮

**编辑员工**：
1. 点击员工行的"编辑"按钮
2. 修改员工信息
3. 点击"确定"按钮保存

**删除员工**：
1. 点击员工行的"删除"按钮
2. 在确认对话框中点击"确定"

**查看绩效**：
1. 点击员工行的"绩效"按钮
2. 查看员工的工作统计和效率分析

### 4. 任务分配

#### 4.1 功能说明
任务分配模块提供任务的智能分配功能，支持拖拽分配、批量分配等操作。

#### 4.2 分配方式
**拖拽分配**：
1. 在待分配任务列表中找到要分配的任务
2. 拖拽任务到目标员工区域
3. 系统自动完成分配

**手动分配**：
1. 点击任务行的"分配"按钮
2. 在弹出的对话框中选择员工
3. 设置优先级和备注
4. 点击"确定"按钮

**批量分配**：
1. 选择多个待分配任务
2. 点击"批量分配"按钮
3. 选择目标员工
4. 点击"确定"按钮

#### 4.3 分配规则
- 优先分配给岗位匹配的员工
- 考虑员工当前任务数量
- 考虑任务优先级
- 考虑员工技能等级

### 5. 效率分析

#### 5.1 功能说明
效率分析模块提供厨房运营效率的全面分析，包括时间趋势、完成量、岗位对比等。

#### 5.2 核心指标
- **平均准备时间**: 任务从分配到开始制作的平均时间
- **平均完成时间**: 任务从开始到完成的平均时间
- **任务完成率**: 已完成任务占总任务的比例
- **准时完成率**: 按时完成任务占总任务的比例

#### 5.3 图表分析
**订单处理时间趋势**：
- 显示每日平均处理时间变化
- 帮助识别效率波动
- 支持按日期范围筛选

**任务完成量趋势**：
- 显示每日完成任务数量
- 帮助分析工作负荷
- 识别高峰时段

**各岗位效率对比**：
- 对比不同岗位的效率
- 识别效率瓶颈
- 优化人员配置

**高峰时段分析**：
- 显示各时段任务数量
- 识别业务高峰
- 优化排班安排

#### 5.4 效率报告
系统自动生成效率分析报告，包括：
- 关键指标汇总
- 趋势分析
- 问题识别
- 优化建议

#### 6.1 功能说明

#### 6.2 监控内容

4. 保存修改

**删除排班**：
1. 找到要删除的排班

#### 7.3 排班模板

- 快速应用排班模板
- 提高排班效率

## 常见问题

### Q1: 如何快速分配任务？
A: 使用拖拽功能，将待分配任务直接拖到目标员工区域，系统会自动完成分配。

### Q2: 如何查看员工绩效？
A: 点击员工行的"绩效"按钮，可以查看员工的详细绩效数据，包括完成任务数、效率评分等。

### Q3: 如何优化厨房效率？
A: 查看效率分析模块，系统会根据数据分析提供优化建议，如优化高峰期人员配置、提升某岗位效率等。

### Q4: 如何设置库存预警？
A: 在库存管理中，为每个食材设置最低库存量和保质期，系统会在库存不足或即将过期时自动预警。

### Q5: 如何处理超时任务？
A: 系统会自动识别超时任务并发送告警，可以在任务管理中查看超时任务，分析原因并优化流程。

### Q6: 如何创建排班模板？
A: 在排班管理中，设置好一次完整的排班后，点击"保存为模板"，下次可以直接应用该模板。

### Q7: 如何导出运营报表？
A: 在报表导出模块中，选择报表类型和导出格式，设置时间范围，点击"导出"按钮即可。

### Q8: 如何查看设备维护记录？
A: 在设备管理中，点击设备行的"详情"按钮，可以查看设备的完整维护记录。
