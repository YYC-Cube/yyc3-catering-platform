# YYC³餐饮行业智能化平台 - 导航栏菜单文档

## 1. 管理后台 (Admin Dashboard)

### 1.1 侧边栏主菜单

| 菜单名称 | 页面功能组件描述 |
|---------|----------------|
| **工作台** | 管理后台的首页仪表板，展示核心业务指标、实时数据概览和快捷操作入口 |
| **订单管理** | 管理所有订单信息，包括订单列表、订单详情、订单状态管理、订单搜索和筛选 |
| **菜单管理** | 管理餐厅菜单，包括菜品分类、菜品信息、价格管理、上架/下架操作 |
| **用户管理** | 管理平台注册用户，包括用户列表、用户详情、权限设置和用户数据统计 |
| **员工管理** | 管理餐厅员工，包括员工信息、职位管理、考勤记录和权限分配 |
| **财务管理** | 管理财务数据，包括收支记录、账单管理、报表生成和财务分析 |
| **数据分析** | 提供业务数据的可视化分析，包括销售趋势、用户行为、菜品热销排行等 |
| **系统设置** | 配置平台系统参数，包括基础设置、通知设置、权限管理和系统维护 |
| **营销管理** | 管理营销活动，包括优惠券、促销活动、会员积分和营销效果分析 |
| **安全审计** | 监控系统安全状态，包括操作日志、异常行为检测和安全策略配置 |
| **帮助中心** | 提供系统使用帮助文档、常见问题解答和技术支持入口 |

## 4. 导航栏功能特性

### 4.1 响应式设计

- 适配不同屏幕尺寸，在桌面端、平板和移动端提供优化的导航体验
- 移动端自动切换为底部导航栏，提高操作便捷性

### 4.2 实时通知

- 新订单通知、预约提醒、系统消息等实时推送
- 未读消息数量提示，支持快速查看和处理

### 4.3 快捷操作

- 提供常用功能的快捷入口，如快速下单、桌台状态查看等
- 支持自定义快捷操作，提升工作效率

### 4.4 权限控制

- 根据用户角色和权限动态显示可用菜单
- 确保不同用户只能访问其权限范围内的功能

### 4.5 状态指示

- 显示用户当前状态（如在线/离线、工作中/休息中）
- 订单状态、桌台状态等实时更新和显示

## 5. 使用说明

### 5.1 导航操作

- **桌面端**：使用侧边栏或顶部导航菜单进行页面切换
- **移动端**：使用底部导航栏或顶部菜单图标打开导航菜单

### 5.2 菜单搜索

- 在管理后台和员工端提供菜单搜索功能
- 支持按菜单名称、功能描述等关键词搜索

### 5.3 自定义导航

- 管理后台支持自定义菜单顺序和显示
- 员工端和客户端支持个性化导航设置
